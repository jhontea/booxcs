MEMBUAT LOGIN UNTUK MULTI USER DI CODEIGNITER

Kali ini saya akan memberikan tutorial cara membuat halaman login untuk multi user. Contohnya kali ini saya akan memberikan contoh pada sistem informasi percetakan yang diakses untuk 3 macam user, yaitu user penulis, user editor, serta user untuk toko. Tiap tiap user akan masuk pada halaman yang berbeda. Pada kali ini saya menggunakan tamplate botstrap dimana semua sudah tersedia dan kita tinggal mengedit sesuai kebutuhan kita.
STEP 1 : Membuat tabel
Sobat diasumsikan sudah membuat database bernama percetakan dan membuat tabel bernama user dengan 5 field/kolom yaitu id_user, nama, email, username, password, dengan field id kita set sebagai primary key dengan nilai auto_increment.

<?php
 
Class User extends CI_Model
 
{
 
function login($username, $password)
 
{
 
$this -&gt; db -&gt; select('*');
 
$this -&gt; db -&gt; from('user');
 
$this -&gt; db -&gt; where('username', $username);
 
$this -&gt; db -&gt; where('password', $password);
 
//$this -&gt; db -&gt; limit(1);
 
$query = $this -&gt; db -&gt; get();
 
if($query -&gt; num_rows() == 1)
 
{
 
return $query-&gt;result();
 
}
 
else
 
{
 
return false;
 
}
 
}
 
}
 
?>

function check_database($password)
 
{
 
$username = $this->input->post('username');
 
$result = $this->user->login($username, $password);
 
&nbsp;
 
if($result)
 
{
 
$sess_array = array();
 
foreach ($result as $row ) {
 
$sess_array = array(
 
'id' => $row->id_user,
 
'username' => $row->username,
 
'level_user' => $row->level_user
 
);
 
$this->session->set_userdata('logged_in', $sess_array);
 
}
 
return TRUE;
 
}
 
else{
 
$this->form_validation->set_message('check_database', 'Invalid username or password');
 
return false;
 
}
 
}
 
}

<?php if ( ! defined('BASEPATH')) exit('No direct script access allowed');
session_start();
class About extends CI_Controller {
 
 function __construct()
 {
   parent::__construct();
 }
 
 function index()
 {
    if($this->session->userdata('logged_in')){
     $this->load->model('user');
     $this->load->model('profile');
     $session_data = $this->session->userdata('logged_in');
     $id_user=$session_data['id'];
     $var['username'] = $session_data['username'];
    if($session_data['level_user']== "1"){
          $this->load->view('editor_view')
        }elseif($session_data['level_user'] == "2"){
          $this->load->view('penulis_view')
        }elseif($session_data['level_user'] == "3"){
          $this->load->view('toko_view')
        }
 
     $this->load->view('editor_view',$var);
   }else{
    redirect('welcome', 'refresh');
   }
 
 }
 
 function logout()
 {
   $this->session->unset_userdata('logged_in');
   session_destroy();
   redirect('welcome', 'refresh');
 }
}
 
?>

setelah membuat itu semua harusnya kita membuat halaman view serta controller untuk tiap tiap menu yang ada pada
tiap tiap user. akan tetapi dalam tutorial ini hanya menjelaskan menu login. untuk lebih jelasnya kita bahas
runtutan cara kerja dalam menu login ini.

pertama dimulai dari login_view.php dalam login view ini kita kan mengisikan username dan password pada apabila password
dan username belum diisi akan tetapi sudah login akan muncul tanda peringatan. setelah username dan password
diisikan lalu tombol login memverifikasikan username dan password apakah benar atau salah. dalam hal ini langsung
menuju pada verifylogin.php yang ada pada controller.di verify login ini username dan password disamakan dengan
data yang ada pada database. apabila sudah masuk sesuai username dan password halaman ini mengambil user level dimana
tiap user level yang menemtukan apakah user ini penulis, toko , atau editor. dan setelah masuk pada itu akan masuk
pada halaman awal user masing masing.
